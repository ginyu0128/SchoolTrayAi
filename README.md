# SchoolTrayAI

AI-based school tray nutrition estimation prototype.

## Features
- Upload a tray photo
- Detect tray and segment tray cells
- Estimate food volume and calories
- Integrate with public nutrition data API
- Provide a Streamlit UI

## Run
```powershell
python -m streamlit run app.py
```

Default local URL:
```text
http://localhost:128
```
